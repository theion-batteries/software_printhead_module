
=================================================================================================
    Getting static Xerox W-series head information
=================================================================================================

Static information, like head serial number etc. is extracted from head NVRAM and published as "EEPROM data".
This information can be obtained in a form of JSON file via PiGetEepromData() API.

This information can also be obtained via TestApp. "Status" tab -> "Get Head EEPROM data" button.
Note that correct PCC/HDC numbers needs to be specified in the dropdown lists on the left.
