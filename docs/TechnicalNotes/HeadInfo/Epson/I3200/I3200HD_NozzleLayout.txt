=================================================================================================
    Some information on nozzle layout for the Epson I3200-A1HD head
=================================================================================================

This document shows head nozzle layout from our HDC point of view.

Important! The picture below if a view THROUGH the head on to the paper.

           C1  C2       C3  C4       C4  C3       C2  C1   <- Head colour index (the way ink is plumbed in the head)
            |  |         |  |         |  |         |  |
            |  |         |  `---------'  |         |  |
            |  |         `---------------'         |  |
            |  `-----------------------------------'  |
            `-----------------------------------------'


There are 4 ink ports in this head. They map to Meteor colour planes by default as follows: 

    ink PortA   -> rows A,H -> C1 -> Head mapping *:1:1
    ink PortB   -> rows B,G -> C2 -> Head mapping *:1:2
    ink PortC   -> rows C,F -> C3 -> Head mapping *:1:3
    ink PortD   -> rows D,E -> C4 -> Head mapping *:1:4

There are 8 ink ports in this head. They map to Meteor colour planes by default as follows: 

    ink PortA   -> row A -> C1 -> Head mapping *:1:1
    ink PortB   -> row B -> C2 -> Head mapping *:1:2
    ink PortC   -> row C -> C3 -> Head mapping *:1:3
    ink PortD   -> row D -> C4 -> Head mapping *:1:4
    ink PortE   -> row E -> C5 -> Head mapping *:1:5
    ink PortF   -> row F -> C6 -> Head mapping *:1:6
    ink PortG   -> row G -> C7 -> Head mapping *:1:7
    ink PortH   -> row H -> C8 -> Head mapping *:1:8
